<h1>Projeto</h1>
Jogo da memória

<h1>Tecnologias</h1>
- ReactJS
- Typescript
- Styled-Components

<h1>Como executar</h1>
- Clone o repositório
- Instale as dependências com npm install
- Inicie o servidor com npm start

Agora você pode acessar localhost:3000 do seu navegador

Divirta-se!
